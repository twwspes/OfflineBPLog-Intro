Rationale behind the application creation

The app was created out of my mother's need for recording her blood pressure records, but the current blood pressure log applications available on app stores (Apple App Store / Google Play) were either too complex for her to use or requiring registration. Therefore, this application does only one thing, simply recording user's everyday blood pressure readings. The blood pressure records are only stored in the users' device so that no one except the user can view them.

Another reason for the application's creation is that I, as a web / mobile developer, need constantly create and modify applications to hone my coding skills and keep abreast of the latest development of application technologies.
